# AI 활용 예술공모 워크숍 - 4주차 프레젠테이션

## 📖 개요
**AI 언어모델(제미나이) 기본 사용법**을 주제로 한 120분 강의 자료입니다.
Reveal.js 기반으로 제작된 모던하고 인터랙티브한 웹 프레젠테이션입니다.

---

## ✨ 주요 기능

### 🎨 디자인
- **예술적 그라디언트 스타일**: 보라-핑크 계열의 부드러운 그라디언트
- **프리미엄 UI**: 둥근 모서리, 섬세한 그림자, 현대적인 레이아웃
- **가독성 최적화**: Noto Sans KR 폰트, 명확한 시각적 계층
- **반응형**: 다양한 화면 크기에 대응

### 🧭 네비게이션
- **상단 네비게이션 바**: 주요 섹션으로 빠른 이동
  - 도입
  - 이론
  - 프롬프트
  - 실습
  - 마무리
- **진행률 표시**: 하단 프로그레스 바로 현재 위치 확인
- **키보드 조작**: 화살표 키 또는 클릭으로 슬라이드 이동

### 📄 PDF 저장
- **PDF 버튼**: 상단 우측의 "PDF 저장" 버튼 클릭
- **안내 모달**: URL에 `?print-pdf` 추가 방법 안내
- **간편한 변환**: 브라우저 인쇄 기능 (Ctrl+P)으로 PDF 저장

## 🚀 사용 방법

### 1. 프레젠테이션 열기
```
index.html 파일을 브라우저에서 열기
(더블클릭 또는 브라우저에 드래그)
```

### 2. 슬라이드 조작
- **다음 슬라이드**: 오른쪽 화살표 키 (→) 또는 화면 우측 클릭
- **이전 슬라이드**: 왼쪽 화살표 키 (←) 또는 화면 좌측 클릭
- **섹션 이동**: 상단 네비게이션 바의 버튼 클릭
- **개요 보기**: ESC 키
- **전체 화면**: F11 또는 F키

### 3. PDF로 저장하기

#### 방법 1: 상단 버튼 사용
1. 상단 우측 "PDF 저장" 버튼 클릭
2. 모달에서 "URL 복사하기" 버튼 클릭
3. 복사된 URL을 새 탭에 붙여넣기
4. Ctrl+P (인쇄) → "PDF로 저장" 선택

#### 방법 2: 직접 입력
1. 브라우저 주소창에서 URL 끝에 `?print-pdf` 추가
   예: `file:///e:/developement/ai-class-ppt/index.html?print-pdf`
2. Ctrl+P → "PDF로 저장" 선택
